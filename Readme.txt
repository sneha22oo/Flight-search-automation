PROBLEM STATEMENT : 
====================  

IRCTC Flight Search automation 

Get the list of flights available today from Hyderabad to Pune. 

Suggested site: https://www.air.irctc.co.in/ 


Detailed description: 
======================

1. Launch https://www.air.irctc.co.in/ in a web browser, and verify the appropriate site is opened or not. 

2. Maximize the window and verify the respective application is opened or not 

3. Enter "Hyd" in “From” city, in the auto listed results, select "HYDERABAD (HYD)"  

4. Enter "Pune" in "To" city, in the auto listed results, select "PUNE (PNQ)"  

5. Select today’s date from the date picker. 

6. Select "Business" class, from “Choose class” dropdown box in Traveller details field. 

7. Click on "Search" button. 

8. Verify the results shown are valid, by checking the city and date values are same as given in previous page. 

9. Display the name and Number of available Flights on the console 

10. Capture the results screenshot and save in the project folder. 

11. Close the Application. 

KEY AUTOMATION SCOPE :
======================

* Multi Browser automation (Chrome and Firefox)
* Extract multiple options items & store in arrays or collections
* Filling form (in different objects in web page)
* Capture results, warning or error messages
* Handle bootstrap dropdown elements
* Validating results
